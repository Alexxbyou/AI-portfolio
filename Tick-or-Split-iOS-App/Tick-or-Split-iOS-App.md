# Tick or Split iOS App

## Problem Statement
- Simple mobile application to assist in personal time management, tracking and prioritization of tasks.
- An implementation of the "Tick or Split" methodology for task management, which focuses on the execution of tasks in a structured manner, allowing users to "tick" completed tasks and "split" larger tasks into smaller, manageable sub-tasks.


## Solution Design
- "Tick or Split" methodology that focuses on the execution and the tracking of tasks in a structured manner.
- "Tick or Split" app that realize the core principles of the methodology, with app features.


## Tech Stack
- Mobile Application Development: iOS (Swift / SwiftUI)
- Development Support: OpenAI Codex, Claude Code
