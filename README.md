## Report Structure
Each project report includes the following sections:
- **Problem Statement:** A clear definition of the problem being addressed.
- **Solution Design:** An outline of the proposed solution, including key components and architecture.
- **Tech Stack:** A list of technologies and tools used in the project.
- **Future Applicability:** A discussion of how the solution can be applied in future contexts or scaled to other domains.
- **Relevant AI Use Case Category:** A categorization of the project based on its primary AI use case, such as Information Search, AI Augmented Product, or AI Coworker.
- **Github Repository:** A link to the project's code repository for further exploration and reference.
