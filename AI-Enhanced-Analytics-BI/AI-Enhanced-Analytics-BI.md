# AI Enhanced Analytics & BI (WIP)

## Problem Statement
- To build an agentic analytics and BI system that can dynamically extract, analyze, visualize, and generate insights from complex data sources, including databases, spreadsheets, and unstructured data.
- The system should be able to handle a wide range of analytical tasks, from simple queries to complex data transformations and visualizations, while maintaining a user-friendly interface for non-technical users.

## Solution Design
- An agentic architecture that combines LLMs with data processing and visualization tools to create a seamless analytics experience.
- Agent tools to interpret and disect user queries, determine the output format, subqueruies, and data sources needed to fulfill the request.
- Agent tools to convert natural language queries into executable data queries, perform data ETL.
- Agent tools to generate visualizations and insights based on the processed data.
- Agent tools to consolidate and summarize insights into a coherent response for the user.

## Tech Stack
- LLM & Orchestration: RAG (Retrieval-Augmented Generation), LangGraph, DSPy, LangChain
- Data Processing: SQL, Pandas
- Visualization: Plotly

## Future Applicability
- Enables natural language → SQL → visualization workflows to complement traditional BI reporting
- Has the potential to integrate into existing reporting ecosystems to surface new insights and iteratively refine recurring reporting needs
